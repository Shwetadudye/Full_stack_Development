HTML TAGS

Heading Tags

  • Html provides 6 types of headings : h1 to h6
  • h1 is the largest and most important heading
  • h6 is the smallest heading
  • Headings define the structure of the webpage
  • Very important for SEO (Search Engine Optimization)

<h1>This is first heading</h1>

<h1>This is my second heading</h1>

<h1>This is my third heading</h1>

<h1>This is my forth heading</h1>
<h1>This is my fifth heading</h1>
<h1>This is my sixth heading</h1>

Paragraph Tag

  • The <p> tag defines a paragraph
  • Browsers automatically add some space (margin) before and after a paragraph
  • Paragraphs are used to structure text content on a webpage

<p>This is my first paragraph.</p>

<p>This is my second paragraph.</p>

Break Tag

  • The <br> tag inserts a single line break
  • It is an empty tag, meaning it does not have a closing tag
  • Useful for writing addresses or poems where line breaks are significant

This is my first line.<br>This is my second line.

Horizontal Rule Tag

  • The <hr> tag defines a thematic break in an HTML page
  • It is used to separate content or sections visually
  • It is an empty tag, meaning it does not have a closing tag

This is some text above the horizontal line.<hr>This is some text below the horizontal line.

Pre Tag

  • The <pre> tag defines preformatted text
  • Text inside a <pre> element is displayed in a fixed-width font
  • Whitespace inside a <pre> element is displayed as written in the HTML code 

 <pre>

        This     is   my preformatted   
        
        text.        It preserves both   spaces
        and line breaks.

        </pre>

Mark Tag

  • <mark> tag is used to highlight text.
  • Default colour is yellow
  • Used to marking important or searched text
  • It is an inline element
Hello <mark>Shweta </mark> dudye

Image with alt

  • <img> tag is used to display images
  • alt attribute : text shown if images fails to load
<img src=".jpg" alt=Image_name >
