HTML FORMATTING TAGS

Bold & Strong

  • <b> makes text bold (visual only)
  • <strong> makes text bold + gives importance (semantic)
<b>Hello World</b> → Hello World
<strong>Important Text</strong> → Important Text

Italic & Emphasis

  • <i> italic text (visual only)
  • <em> italic + stress emphasis (semantic)
<i>Italic Text</i> → Italic Text
<em>Important Text</em> → Important Text

Underline

  • <u> underlines text
  • Used for highlighting names or references
<u>Shweta Dudye</u> → Shweta Dudye

Small & Big Text

  • <small> makes text smaller
  • <big> makes text bigger (deprecated but still works)
<small>Small Text</small> → Small Text
<big>Big Text</big> → Big Text

Deleted & Inserted Text

  • <del> shows deleted text (strikethrough)
  • <ins> shows inserted text (underlined)
<del>Old Price</del> → Old Price
<ins>New Price</ins> → New Price

Subscript & Superscript

  • <sub> = subscript (below baseline)
  • <sup> = superscript (above baseline)
H<sub>2</sub>O → H2O
2<sup>3</sup> → 23

Mark Tag

  • <mark> highlights text (yellow)
  • Inline element
<mark>Highlighted Text</mark> → Highlighted Text
