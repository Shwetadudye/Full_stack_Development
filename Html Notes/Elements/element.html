HTML5 Elements

An HTML element is the building block of a webpage.
It includes :

  • A Start Tag
  • Content
  • An End Tag

Example of an HTML Element :

ELEMENT = <p>+content </p>
  • Tag : only the <p> and </p> parts are tags.
  • Element : The start tag, content, and end tag together make up an element.

Types of Elements

Element Type Description Example
Paired Elements These elements have opening and closing tags. <div>, <p>, <h1>
Empty (Self closing ) Elements These elements do not have an end tag <br>, <hr>, <img> <input>,<meta>,<link>
Block -Level Elements Take Full width of the page. Always start on a new line <div>, <p>, <h1>, <section>,<article>
Inline Elements Take only as much width as necessary. Do not start on a new line <span>, <a>, <img>, <strong>,<em>

Semantic and Non Semantic Elements

Element Type Description Example
Semantic Elements These elements clearly describe their meaning to both the browser and the developer. <article>, <section>, <header>, <footer>, <nav>
Non Semantic Elements These elements do not provide any information about their content. <div>, <span>
