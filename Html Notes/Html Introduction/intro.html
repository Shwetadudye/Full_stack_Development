INTRODUCTION OF HTML

Html , which is stands for "Hypertext markup language" the foundational language of the web. It's used to create and struture websites.By utilizing Html tags , we can define the appearance and layout of a website . With a good grasp of these tages and their proper usage,creating beautiful websites becomes straightforward and efficient

THEN WHY CSS AND JAVASCRIPT

  • Html is used to define the layout of a page , providing a barebone structure for the content.
  • Css is used to add styling to that barebone page created using HTML.
  • Javascript is used to program logic for the page layout e.g. What happen when user hovers on the text , whn to hide or show elements etc

A BEAUTIFUL ANALOGY

  • Html : Human body (Skeleton)
  • Css : Presentation
  • Javascript : Behaviour
