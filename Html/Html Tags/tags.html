HTML TAGS

Heading Tags

  • HTML provides 6 types of headings: h1 to h6
  • h1 is the largest and most important heading
  • h6 is the smallest heading
  • Headings define the structure of the webpage
  • Very important for SEO (Search Engine Optimization)

<h1>This is first heading</h1>

<h2>This is my second heading</h2>

<h3>This is my third heading</h3>

<h4>This is my fourth heading</h4>

<h5>This is my fifth heading</h5>
<h6>This is my sixth heading</h6>

Paragraph Tag

  • The <p> tag defines a paragraph
  • Browsers automatically add margin before and after a paragraph
  • Paragraphs are used to structure text content
<p>This is my first paragraph.</p>

<p>This is my second paragraph.</p>

Break Tag

  • The <br> tag inserts a single line break
  • It is an empty tag (no closing tag)
  • Useful for addresses or poems where line breaks matter
This is my first line.<br>This is my second line.

Horizontal Rule Tag

  • The <hr> tag defines a thematic break
  • Used to separate content visually
  • It is an empty tag (no closing tag)
This is text above the line.<hr>This is text below the line.

Pre Tag

  • The <pre> tag defines preformatted text
  • Displayed in a fixed-width font
  • Whitespace and line breaks are preserved
<pre>
This     is   preformatted  
text. It preserves    spaces 
and line breaks.
</pre>

Mark Tag

  • <mark> tag is used to highlight text
  • Default highlight color is yellow
  • Inline element
Hello <mark>Shweta</mark> Dudye

Image with alt

  • <img> tag is used to display images
  • alt text shows when image fails to load
<img src="image.jpg" alt="Image_name">
