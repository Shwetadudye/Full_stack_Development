Difference between html amd html5

HTML HTML5
Audio and video are not part of it. Audio and video are integral part of Html5.
HTML is less Mobile friendly. HTML5 is more mobile friendly.
HTML uses tags like <div>, <span> for structuring content. HTML5 introduces new semantic elements like <header>, <footer>, <article>, <section> for better content structure.
Html Not Support Drag and Drop effect. Html5 Support Drag and Drop effect.
Html not supported Audio and Video without using flash player. HTML5 Support Audio and Video with use of <audio> and <video> tag.
HTML Not Support Javascript to run in browser. HTML5 Support Javascript to run in background with the help of j.s-web worker API.
HTML does not support local storage. HTML5 supports local storage using Web Storage API (localStorage and sessionStorage).
Html Cannot handle inaccurate syntax Html5 Can handle inaccurate syntax
Shapes like Circle, Rectangle ,Triangle etc are not possible to draw Shapes like Circle, Rectangle ,Triangle etc are possible to draw using <canvas> element
