HTML STYLES

What is style attribute?

  • Used to apply CSS inside HTML element
  • Written using the style="" attribute
  • Used for inline styling
  • The HTML style attribute is used to add styles to an element, such as color, font, size, and more.
  • Basic syntax:
    <tag style="property:value;">
Example:
<p style="color:red;">Hello</p>
→

Hello

Style: Color

  • Changes text color
  • Values: color name, hex, rgb
<p style="color:blue">Blue Text</p>
→

Blue Text

Style: Background Color

  • Changes background color of element
<div style="background-color:yellow">Content</div>
→
Content

Style: Font Size

  • Changes size of text
  • Values: px, em, rem, %
<p style="font-size:25px">Large Text</p>
→

Large Text

Style: Font Family

  • Changes the font of text
  • Multiple fonts can be used as fallback
<p style="font-family:Arial">Sample Text</p>
→

Sample Text

Style: Text Align

  • Aligns text inside an element
  • Values: left, right, center, justify
<p style="text-align:center">Centered Text</p>
→

Centered Text

Style: Border

  • Adds border around an element
  • Format: width | style | color
<p style="border:2px solid red">Paragraph</p>
