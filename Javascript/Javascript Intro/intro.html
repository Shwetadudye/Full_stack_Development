What is JavaScript?

  • High-level, dynamic, interpreted language.
  • Used to make webpages interactive.
  • Runs in browsers and on servers (Node.js).

Why JavaScript?

  • Adds functionality to webpages.
  • Handles user events (click, keypress).
  • Used for form validation.
  • DOM manipulation (change HTML/CSS).

Ways to Add JavaScript

<script> console.log("Hi"); </script>
<script src="app.js"></script>
<button onclick="alert('Hello')">Click</button>

Basic Syntax

  • Statements end with semicolon (optional).
  • Case-sensitive (x and X are different).
  • Uses comments:
// Single-line
/* Multi-line */

Output Methods

  • console.log()
  • alert()
  • document.write()
  • innerHTML

Variables

var   // old
let   // block-level
const // constant

Data Types

  • Primitive: String, Number, Boolean, Null, Undefined, BigInt, Symbol
  • Non-Primitive: Object, Array, Function
