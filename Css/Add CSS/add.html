How to Add CSS in HTML (Notes)

1️⃣ Inline CSS

CSS added directly inside the HTML element using the style attribute.

<p style="color: red; font-size: 20px;"> This is inline CSS </p>

Use when: You need quick, one-line styling (not for big projects).

2️⃣ Internal CSS

CSS written inside the <style> tag in the <head> section of the HTML page.

<head> <style> h1 { color: blue; text-align: center; } </style> </head> <h1>This uses internal CSS</h1>

Use when: You want styling for one single HTML file.

3️⃣ External CSS

CSS written in a separate .css file and linked to HTML using the <link> tag.

/* style.css */ h2 { color: green; font-size: 28px; } <head> <link rel="stylesheet" href="style.css"> </head> <h2>This uses external CSS</h2>

Use when: You want clean code, reusable styles, and professional projects.
