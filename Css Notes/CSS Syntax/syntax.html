CSS3 Syntax Notes (English)

1️⃣ CSS Basic Syntax

CSS syntax is made up of three parts: selector, property, and value.

selector { property: value; }

Example:

h1 { color: blue; font-size: 30px; }

2️⃣ Declaration Block

All CSS rules written inside { } are called the declaration block.

p { color: green; text-align: center; }

Each declaration must end with a semicolon ;.

3️⃣ Types of CSS Selectors

Element Selector:

p { color: red; }

Class Selector:

.box { background: yellow; }

ID Selector:

#header { padding: 20px; }

Group Selector:

h1, h2, p { color: blue; }

Universal Selector:

* { margin: 0; padding: 0; }

4️⃣ Important CSS3 Features

Border Radius:

border-radius: 10px;

Box Shadow:

box-shadow: 0 0 10px gray;

Text Shadow:

text-shadow: 2px 2px 5px black;

Transition:

transition: all 0.3s ease;

Transform:

transform: rotate(45deg);

Animation:

animation: move 3s infinite;

Flexbox:

display: flex;

Grid:

display: grid;

5️⃣ CSS Comments

/* This is a comment */

6️⃣ Ways to Apply CSS in HTML

External CSS:

<link rel="stylesheet" href="style.css">

Internal CSS:

<style> p { color: blue; } </style>

Inline CSS:

<p style="color:red;">Hello</p>
