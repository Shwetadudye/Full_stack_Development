CSS3 Syntax Notes

1️⃣ CSS Basic Syntax

CSS syntax is made up of three parts: selector, property, and value.

selector { property: value; }

Example:

h1 { color: blue; font-size: 30px; }

2️⃣ Declaration Block

All CSS rules written inside { } are called the declaration block.

p { color: green; text-align: center; }

Each declaration must end with a semicolon ;.

3️⃣ Types of CSS Selectors

Element Selector:

p { color: red; }

Class Selector:

.box { background: yellow; }

ID Selector:

#header { padding: 20px; }

Group Selector:

h1, h2, p { color: blue; }

Universal Selector:

* { margin: 0; padding: 0; }
