CSS Colors Notes

1️⃣ Using Color Names

CSS provides predefined color names like red, blue, green, yellow, etc.

color: red; background-color: lightblue; border: 2px solid green;

2️⃣ Hex Color Codes

Hex codes define color values using a 6-digit combination of numbers and letters.

color: #ff0000; /* red */ color: #00ff00; /* green */ color: #0000ff; /* blue */

Shortcut (3-digit Hex):

color: #f00; /* red */ color: #0f0; /* green */ color: #00f; /* blue */

3️⃣ RGB Colors

RGB stands for Red, Green, Blue (0–255 range).

color: rgb(255, 0, 0); /* red */ color: rgb(0, 255, 0); /* green */ color: rgb(0, 0, 255); /* blue */

4️⃣ RGBA Colors (with Transparency)

A is Alpha → controls opacity (0 = transparent, 1 = solid).

color: rgba(255, 0, 0, 0.5); /* 50% transparent red */ color: rgba(0, 0, 255, 0.2); /* light transparent blue */

5️⃣ HSL Colors

HSL = Hue, Saturation, Lightness

color: hsl(0, 100%, 50%); /* red */ color: hsl(120, 100%, 50%); /* green */ color: hsl(240, 100%, 50%); /* blue */

6️⃣ HSLA (with transparency)

color: hsla(0, 100%, 50%, 0.4); /* transparent red */

7️⃣ Gradient Colors

Used for smooth transitions between two or more colors.

background: linear-gradient(red, yellow); background: linear-gradient(to right, blue, purple); background: radial-gradient(circle, red, orange, yellow);

8️⃣ Quick Summary (Revision)

Color Names: red, blue, green Hex: #ff0000 RGB: rgb(255, 0, 0) RGBA: rgba(255, 0, 0, 0.5) HSL: hsl(0, 100%, 50%) HSLA: hsla(0, 100%, 50%, 0.5) Gradients: linear-gradient(), radial-gradient()

Best Practice: Use HEX or RGB for most projects.
